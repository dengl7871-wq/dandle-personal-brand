# 国内静态版部署说明

这个项目已经支持生成纯静态版本，适合部署到国内访问更稳定的平台，例如：

- 腾讯云 EdgeOne Pages
- 腾讯云 COS 静态网站
- 阿里云 OSS 静态网站
- 其他任意支持上传静态文件的国内服务器或对象存储

## 生成国内版

在项目根目录运行：

```bash
npm run build:cn
```

生成结果在：

```bash
out/
```

把 `out/` 目录里的所有文件上传到国内静态网站服务即可。首页文件是 `index.html`。

## 推荐部署方式

### 方案 A：腾讯云 EdgeOne Pages

适合想要省心、自动化部署的人。把 GitHub 仓库连接到 EdgeOne Pages，构建命令填：

```bash
npm run build:cn
```

输出目录填：

```bash
out
```

### 方案 B：腾讯云 COS / 阿里云 OSS

适合手动上传。先运行 `npm run build:cn`，再把 `out/` 目录里的内容上传到 Bucket，并开启静态网站托管。

## 域名建议

如果只是临时投递学校招聘，可以先使用平台提供的访问域名。

如果要长期正式使用国内服务器或对象存储，并绑定自己的域名，通常需要准备 ICP 备案。
